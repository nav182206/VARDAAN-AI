📖 Vardaan AI - Your Personalized Academic Partner
Vardaan AI is an elite, Socratic academic coach designed specifically for Class 11-12 students following the NCERT curriculum in India. Unlike traditional AI tutors that simply provide answers, Vardaan acts as a "Phantom Step" detector—identifying unstated assumptions, logical leaps, and hidden misunderstandings in a student's reasoning.

✨ Key Features:

🧠 Socratic Coaching & "Phantom Step" Detection: 

Guided Learning: Vardaan uses the Socratic method to lead students to answers through targeted questioning.
Misconception Identification: Specifically designed to catch "Phantom Steps"—common logical errors in Maths, Physics, Chemistry, and Biology.

📊 Mastery & Progress Tracking:

Mastery Dashboard: Visualize your understanding of different topics through interactive charts and graphs.
Study Streak Tracker: Stay motivated with daily streaks and achievement badges.
Smart Goals: Break down high-level goals (like "Crack JEE 2026") into manageable, AI-generated milestones.

🌐 Multilingual & Casual Speech Parsing:

Voice-to-Concept: Speak in your native language (Hindi, Tamil, Telugu, etc.), and Vardaan parses casual intent into precise technical NCERT terminology.
12+ Supported Languages: Including Bengali, Marathi, Gujarati, Kannada, and more.

🏫 Community & Collaboration:

Community Forum: Peer-to-peer doubt resolution with teacher moderation.
Knowledge Vault: A digital repository of AI-generated concept notes and scientific examples.
Role-Based Access: Dedicated dashboards for Students, Teachers, and Admins.

🧘 Student Wellness:

Exam Panic Alert: Instant stress support with AI affirmations and guided breathing cues.
Stress Detection: AI monitors conversation tone to provide emotional support when the pressure gets high.

🚀 Tech Stack:

Frontend: React 19 + Vite
Language: TypeScript
Styling: Tailwind CSS
AI Engine: Google Gemini API (gemini-3-flash & gemini-2.5-flash-image)
Backend/Auth: Supabase
Icons: Lucide React
Charts: Recharts
Math Rendering: KaTeX (via react-katex)

🛠️ Installation & Setup:
1. Clone the repository
code
Bash
git clone https://github.com/your-username/vardaan-ai.git
cd vardaan-ai
2. Install dependencies
code
Bash
npm install
3. Environment Variables
Create a .env file in the root directory and add the following:
code
Env
VITE_GEMINI_API_KEY=your_gemini_api_key_here
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
5. Run the development server
code
Bash
npm run dev

📖 Usage Guide:

Select Your Role: Choose between Student, Teacher, or Admin.
Pick a Subject: Select from Physics, Chemistry, Maths, Biology, CS, or English.
Start a Conversation: Type or use the microphone to ask a doubt.
Review Concept Notes: Check the "Digital Concept Note" sidebar for structured summaries of your chat.
Track Progress: Visit the Mastery Dashboard to see which topics you've conquered.
